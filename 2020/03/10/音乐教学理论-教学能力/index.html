    第一章 音乐课程标准

    一、课程性质

    • 人文性 中国+西方
    • 审美性 美育作用
    • 实践性 听/唱/奏/编创/表演

    记忆：“人家纹身没时间 ”

    二、课程基本理念

    义务教育阶段★★★

    口诀：“审美兴趣创实践，音乐综合多样解。共同发展”

    1. 以音乐审美为核心，以兴趣爱好为动力

      • 审美：美育学科；

        提升审美能力

      • 兴趣：兴趣是音乐学习的根本动力；

        兴趣是人最好的老师；

        事半功倍，提高学习效率。

      • 落实：丰富教学内容；

        多元教学形式。

    2. 强调音乐实践，鼓励音乐创造

      • 实践：

        1）音乐教学本身就是艺术实践的过程；

        2）聆听、演唱、演奏、综合性艺术表演。

      • 创造：通过主观想象，开发创造潜能；培养想象力和创造力。

      • 落实：创设情境；创编。

    3. 突出音乐特点，关注学科综合

      • 音乐特点：时间的艺术、听觉的艺术

        与生活密切相关。

      • 学科综合：VS其他学科；

        使知识形成网络。

      • 落实：注意环节设置（导入、拓展、作业）

        举例。

    4. 弘扬民族音乐，理解音乐文化多样性

      • 民族：中国音乐；

        传承优秀传统文化。

      • 多样性：外国音乐；

        拓宽音乐国际文化视野。

      • 落实：对比欣赏，交流与讨论，合作。

    5. 面向全体学生，注重个性发展

    高中阶段

    1. 以音乐审美为核心，以兴趣爱好为动力
    2. 面向全体学生，注重个性发展
    3. 重视音乐实践，增强创造意识
    4. 弘扬民族音乐，理解多元文化

    三、课程目标

    情感、态度与价值观

    1. 丰富情感体验，培养对生活的积极乐观态度
    2. 培养音乐兴趣，树立终身学习的愿望
    3. 提高音乐审美能力，陶冶高尚情操
    4. 培养爱国主义情感，增强集体主义精神
    5. 尊重艺术，理解世界文化的多样性

    过程与方法

    体验、模仿、探究、合作、综合。

    知识与技能

    音乐基础知识；

    音乐历史与相关文化知识；

    音乐基本技能。

    各学段课程目标要求：

    1-2年级的目标要求：

    1. 激发和培养对音乐的兴趣
    2. 开发音乐的感知力，体验音乐的美感
    3. 自自然地、有表情地演唱，参与其他音乐表现和即兴编创活动。
    4. 培养乐观的态度和友爱精神。

    补充：认识简单音乐知识（四分、八分音符；感受，体验、触摸音乐）。

    3-6年级课程目标：

    1. 保持对音乐的兴趣。
    2. 培养音乐感受与欣赏的能力，初步养成良好的音乐欣赏习惯。
    3. 能自信地、有表情地演唱，乐于参与演奏及其他音乐表现、创造活动。
    4. 培养艺术想象力和创造力。
    5. 培养乐观的态度和友爱精神，增强集体意识，培养合作能力。

    补充：理解基本音乐知识（附点、切分、弱起、记号、6度以上演唱、词曲结合问题等）

    7-9年级的目标要求：

    1. 增进对音乐的兴趣。
    2. 提高音乐感受与评价欣赏的能力，养成良好的音乐欣赏习惯。
    3. 能自信地、有感情地演唱，积极参与演奏及创造活动，发展表现音乐的能力。
    4. 丰富和提高艺术想象力和创造力。
    5. 培养丰富的生活情趣和乐观的态度，增强集体意识，锻炼合作与协调能力。

    补充：理解较深入的音乐知识（复杂的词曲结合问题、简单的曲式、人文知识等）

    高中阶段的目标要求：

    1. 树立终身学习的愿望。
    2. 提高音乐鉴赏和评价的能力，形成健康向上的审美观。
    3. 发展表现音乐的能力。
    4. 增强学生的社会责任感、民族精神和爱国主义情怀；理解和尊重文化的多样性，初步具有国际视野，有助于培养学生参与国际交往的能力。
    5. 增强协作能力和团队意识，培养集体主义精神。

    内容补充：复杂的曲式结构、音乐体裁、音乐要素与情绪情感之间的联系以及丰富的音乐历史知识、人文情怀、音乐的社会功能等。

    四、课程内容

    义务教育“四大领域”

    1.感受与欣赏

    音乐表现要素、音乐情绪与情感、音乐体裁与形式、音乐音乐风格与流派。

    2.表现

    演唱、演奏、综合性艺术表演、识读乐谱

    3.创造

    探索音响与音乐、即兴创编、创作实践

    4.音乐与相关文化

    音乐与社会、音乐姊妹艺术、音乐与艺术之外的其他学科

    高中音乐课程标准

    音乐鉴赏、歌唱、演奏、创作、音乐与舞蹈、音乐与戏剧表演

    • 学分计算：音乐鉴赏：36学时2学分，其余模块18学时1学分。

    五、音乐教学评价

    四个原则：

    1. 导向性原则
    2. 科学性原则
    3. 整体性原则
    4. 可操作性原则

    方式方法：

    1. 形成性评价与终结性评价相结合
    2. 定性评价与定量评价相结合
    3. 自评、互评与他评相结合

    六、课程特点及价值

    1）音乐课程特点

    1. 加强人文教育
    2. 强调课程综合
    3. 注重学生体验
    4. 关注知识更新
    5. 变革学习方式
    6. 培养创新精神
    7. 重建评价机制
    8. 实现民主管理

    2）音乐课程价值★★★

    • 审美体验价值
    • 社会交往价值
    • 创造性发展价值
    • 文化传承价值

    七、当代音乐教育体系

    20世纪音乐教育思想

    （一）多元化教学思想：

    【美】大卫 威廉姆斯——“传统性”“民族性”音乐教育理论（音乐课的主要内容是选用各民族的歌舞、乐曲）

    （二）学科性教学思想：

    1. 【美】尤尼斯 柏德曼——”生产性的音乐学习“理论（”音乐学习不是单个行为训练的过程，而是一个意义生产的过程“。
    2. 【美】爱德文戈登——”*综合性音乐学习**“★★
      • 音乐才能9岁达到”平衡“；
      • 学习顺序：听唱——理解——读谱——创造
      • 基础：培养其音乐的内在听觉

    国外著名音乐教育体系

    （一）达尔克罗兹音乐教学法【瑞士】
    • 1.体态律动：训练身体表达音乐节奏与动态感。
      • 34种元素一览表
      • 人体本身就是一件乐器
      • 顺序：预备、起奏、延续、恢复
      • 核心：节奏训练
    • 2.视唱法 ：
      • 训练耳朵，眼睛和声音来表现音高，旋律和和声。
        • 一线谱
        • 改变速度
        • C自然大调为基础，上行渐强，下行减弱
    • 3.即兴创作：
      • 将音乐元素内化，学生用肢体动作在歌声或者在乐器上进行即兴创作。
    （二）奥尔夫音乐教学法【德国】

    元素性音乐教育

    核心理念：一切从儿童出发，通过亲身实践，主动学习音乐。

    注重主动性、元素性、综合性、创造性的音乐教育。

    • 1.体态律动和节奏练习；

      • 走——1拍
      • 跑——1/2拍

    • 2.声势练习：

      是一种通过有规律地捻指、拍手、拍腿、跺脚等形式掌握各种多变的节奏型，并为幼儿自己创造节奏型积累素材的多元化节奏练习方法。

    （三）柯达伊音乐教学法【匈牙利】

    坚持音乐教育应该立足于本民族的文化精神。

    • 1.首调唱名法：——“移动着的Do”

    • 2.手势唱名法；运用“音阶手势”来定位音高，由匈牙利教育家约翰*柯尔文首创。

    • 3.节奏简记法：柯达伊认为，节奏的训练方法应该尽早采用多声部形式进行。

    • 4.唱名简记法：使用唱名的辅音字头，如d、r、m、f、s、l、t。字母只能 标记音高唱名，不能表示出节奏。

    （四）铃木镇一音乐教学法【日本】
    • 著名”才能音乐教育“，以“母语学习”的方式，开发和培养儿童的音乐潜能。
    • 才能不是天生的，能讲话就能学音乐。强调学习器乐过程中角色的重要性。
    （五）美国综合音乐教学法——发掘创造能力
    • 概况：20世纪60年代产生于美国的一种以探索和即兴创作为核心的“发掘创造力”教学法，即音乐学习的各个方面应当相互联系综合成为一个有机的整体。
    • 基本原则：
      • 音乐共同性因素原则
      • 音乐实践性原则
      • 综合性教学原则（包括全面发展）
    • 课程构建基础：音高（A）、节奏（B）、力度（C）、音色（D）、曲式（E）。
    • 五个环节：自由探索、引导探索、即兴创作、有计划的即兴创作、巩固概念。
